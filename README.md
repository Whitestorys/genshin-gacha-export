# 原神抽卡记录导出工具

Electron + Vite + Vue3 + Tailwindcss

中文 | [English](https://github.com/biuuu/genshin-gacha-export/blob/main/docs/README_EN.md)

# 本地化
修改`src/i18n/`目录下的 json 文件就可以翻译到对应的语言。

## Devlopment

```
# 安装模块
yarn install

# 开发模式
yarn dev

# Build 会根据当前机器平台build出适用版本。
yarn build
```
